<h1>Install software in a Linux distribution</h1>



<h2>Ensure that APT is installed</h2>

- Ensuring APT is installed
- Type APT in the command line
- Confirmed installation of APT

<img src="https://i.imgur.com/MXysTUb.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>Install and uninstall the Suricata application</h2>

- Use the APT package manager to install the Suricata application.
- Type sudo apt install suricata after the command-line prompt and press enter.
- Verify that Suricata is installed by running the newly installed application.
- When Suricata is installed, version and usage information is listed.
- Type suricata after the command-line prompt 


<img src="https://i.imgur.com/mb1V5Zq.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>
<img src="https://i.imgur.com/U6yE7J7.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

- Now to uninstall Suricata
- Type sudo apt remove suricata
- Verify that Suricata has been uninstalled by running the application command again.

<img src="https://i.imgur.com/JnJJo6H.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>Install the tcpdump application</h2>

- Use the APT package manager to install tcpdump.
- Type sudo apt install tcpdump after the command-line prompt

<img src="https://i.imgur.com/6uHzmmh.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>List the installed applications</h2>

- Use the APT package manager to list all installed applications.
- Type apt list --installed
- Search through the list to find the tcpdump application

<img src="https://i.imgur.com/60QJptd.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>
<img src="https://i.imgur.com/MalHCWA.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>Reinstall the Suricata application</h2>

- Run the command to install the Suricata application.
- Type sudo apt install suricata
- Use the APT package manager to list the installed applications.
- Type apt list --installed
- Search through the list to confirm that the Suricata application has been installed.
